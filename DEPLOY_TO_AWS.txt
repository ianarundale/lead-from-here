# AWS Deployment - Quick Reference

## One-Time Setup
```bash
# Install tools
brew install awscli awsebcli

# Configure AWS
aws configure
# Enter: Access Key ID, Secret Key, Region: eu-west-1

# Navigate to project
cd /Users/arundi01/workspace/lead-from-here
```

## Deploy Backend (5-10 minutes)
```bash
# Initialize (first time only)
eb init -p node.js-18 lead-from-here-api --region eu-west-1

# Create environment and deploy
eb create lead-from-here-prod --instance-type t2.micro

# Set environment variable for CORS
eb setenv CORS_ORIGIN="*"

# Get your backend URL (save this!)
eb open
# URL format: http://lead-from-here-prod.eu-west-1.elasticbeanstalk.com
```

## Deploy Frontend (2-5 minutes)
```bash
# Push code to GitHub
git push origin main

# In AWS Console:
# 1. Go to Amplify
# 2. Click "New app" â†’ "Host web app"
# 3. Connect GitHub repository
# 4. Select 'main' branch
# 5. Add environment variable:
#    REACT_APP_BACKEND_URL = http://lead-from-here-prod.eu-west-1.elasticbeanstalk.com

# Amplify auto-deploys after you push!
```

## After Deployment
- Frontend URL: `https://your-app.amplifyapp.com` (see in Amplify console)
- Backend URL: `http://lead-from-here-prod.eu-west-1.elasticbeanstalk.com`
- Test with multiple browser windows
- Share frontend URL with users

## View Logs
```bash
# Backend logs
eb logs --stream

# Check health
eb health

# View all details
eb status
```

## Costs
- **Free tier** (first 12 months with new AWS account)
- After free tier: ~$10-15/month for t2.micro + Amplify

## Tear Down (if needed)
```bash
# Delete Elastic Beanstalk environment
eb terminate lead-from-here-prod

# Delete Amplify app (via AWS Console)
```
